= Сервис работы с задачами

Настоящий документ содержит техническое описание HTTP-сервиса работы с задачами, структура которых приведена на xref:db/db.html[странице].

Сервис позволяет:

. Создавать задачи xref:post/post.adoc [(POST)].
. Получать список созданных задач xref:get-all/get-all.adoc[(GET)].
. Получать конкретную задачу из списка xref:get-all/get-all.adoc[(GET)].
. Изменять параметры задачи (PUT).п
. Удалять задачу (DELETE).
