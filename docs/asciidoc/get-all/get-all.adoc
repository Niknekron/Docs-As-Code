= Получение списка созданных задач (GET)
Метод получения списка созданных задач.

== Входные параметры
Входные параметры отсутствуют.

== Выходные параметры
В случае успешного выполнения метод возвращает:

* список задач с параметрами, указанными в таблице;
* иначе — описание ошибки.

.Параметры задач
[cols="1,1,1,1,2", options="header"]
|===
|Параметр |Ключ |Тип |Не пустой |Описание

|Идентификатор задачи
|task_id
|Число
|true
|Идентификатор задачи.

|Заголовок задачи
|title
|Однострочный текст
|true
|Краткое описание задачи. Длина не более 255 символов.

|Описание
|description
|Многострочный текст
|false
|Полное описание задачи, что необходимо сделать.

|Статус
|status
|Однострочный текст
|true
|Статус исполнения задачи.
|===
