= Создание задачи (POST)
Метод создания задачи.

== Входные параметры
Входные параметры приведены в таблице ниже.

.Входные параметры
[cols="1,1,1,1,2", options="header"]
|===
|Параметр |Ключ |Тип |Не пустой |Описание

|Заголовок
|title
|Однострочный текст
|true
|Краткое описание задачи. Длина не более 255 символов.

|Описание
|description
|Многострочный текст
|false
|Полное описание задачи, что необходимо сделать.
|===

== Выходные параметры
В случае успешного выполнения метод возвращает:

* идентификатор созданной задачи (`task_id`);
* иначе — описание ошибки.
